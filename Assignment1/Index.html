- Levi Hayes Project Idea -

My idea is to have a website which lists a series of miscellanious topics in a way similar to newsvoice.com. They won't have any strong connection between them, but they'll all contain relevant links to the topics, as well as a space for users to chat about said topics and vote their opinions on each topic. The home page will direct users to recent and/or popular polls, using cards, and list out all polls in a more compact space placed lower.
